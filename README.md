# 基于深度学习的照片人脸修复模型的设计与实现

## 安装

- Python >= 3.7
- [PyTorch >= 1.7](https://pytorch.org/)
- Option: NVIDIA GPU + [CUDA](https://developer.nvidia.com/cuda-downloads)
- Option: Linux

### Installation

1. 克隆仓库

    ```bash
    git https://github.com/RandallAnjie/GFPGAN.git
    cd GFPGAN
    ```

1. 安装依赖

    ```bash
    pip install -r requirements.txt
    ```

1. 运行

    ```bash
    python3 main.py
    ```
